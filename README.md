# 🧩 Aplicación de Subastas

## 📌 Descripción
Este proyecto es una aplicación web desarrollada con HTML, CSS y JavaScript que permite crear y visualizar subastas.

## 🚀 Funcionalidades
- Crear subastas
- Validar datos (nombre, precio, fecha)
- Visualizar lista de subastas

## 🛠️ Tecnologías usadas
- HTML
- CSS
- JavaScript

## ▶️ Cómo ejecutar
1. Descargar el proyecto
2. Abrir el archivo index.html en el navegador
